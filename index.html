o que são linguagens de programação de baixo nível e alto nível

Linguagens de alto nível geralmente são mais fáceis de serem usadas em diferentes sistemas como as linguagens java e C++, pois o código escrito nelas tende a ser mais flexível e não depende tanto do tipo de máquina. Já as linguagens de baixo nível estão mais próximas do funcionamento do hardware e, por isso, o código que você escreve pode precisar de ajustes específicos para rodar em cada tipo de sistema, o que torna a portabilidade menor,um exemplo de linguagem de baixo nivel é a Assembly que é uma linguagem de programação de baixo nível que representa comandos em nível simbólico, correspondendo diretamente às instruções do processador.
